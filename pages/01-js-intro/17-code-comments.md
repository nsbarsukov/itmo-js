# Комментарии в коде

<br />

```js
// Этот комментарий занимает всю строку
alert('Привет');

alert('Мир'); // Этот комментарий следует за инструкцией
```

<br />

```js
/* Пример с двумя сообщениями.
Это - многострочный комментарий.
И еще одна строчка комментария.
*/
alert('Привет');
alert('Мир');
```
