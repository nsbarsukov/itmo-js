# Циклы
While

```js
let i = 5;

while (i) {
    console.log(`Обратный отсчет: ${i}...`);
    /**
     * Это так называемый декремент.
     * Уменьшает значение текущей переменной на единицу.
     */
    i--; // i = i - 1
}

// Обратный отсчет: 5...
// Обратный отсчет: 4...
// Обратный отсчет: 3...
// Обратный отсчет: 2...
// Обратный отсчет: 1...
```
