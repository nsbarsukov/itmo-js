# Циклы
for

```js
// формула
for (begin; condition; step) {
    // ... тело цикла ...
}

// пример использования
for (
    let i = 0; // begin
    i < 3; // condition
    i++ // step
) {
    console.log(i);
    // 0
    // 1
    // 2
}
```
