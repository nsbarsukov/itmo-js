# Разновидности функций
Стрелочная функция

```js
// формула
const arrowFunc = (arg1, arg2, ...argN) => expression;

// примеры
const sum = (a, b) => a + b;

const sum = (a, b) => {
    // здесь какая-та многострочная логика
    return a + b;
}

```
